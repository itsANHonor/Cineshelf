1. ACCESS THE APPLICATION
   Open http://localhost:3005 in your browser

2. LOG IN TO ADMIN
   - Go to http://localhost:3005/admin
   - Use your ADMIN_PASSWORD from .env.docker

3. ADD LASERDISC/VHS ITEMS
   - Click "Add New Media"
   - Select "LaserDisc" or "VHS" from format dropdown
   - Fill in details and save

4. TEST CSV EXPORT
   - Go to Admin → Settings tab
   - Find "Import / Export Collection" section
   - Click "Export X Items to CSV"
   - Download will start automatically

5. TEST CSV IMPORT
   - In the same Import/Export section
   - Click "View CSV Schema" to see format
   - Try importing the example-import.csv file:
     * Select "Add to existing collection"
     * Click "Choose CSV File to Import"
     * Select example-import.csv
     * Confirm import

6. VIEW STATISTICS
   - See counts for all 6 formats:
     4K UHD, Blu-ray, DVD, LaserDisc, VHS
   - Stats shown in Media tab and Settings tab

📦 BACKUP YOUR DATA
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Method 1: CSV Export (via UI)
  - Admin → Settings → Export Collection
  - Saves metadata only (not uploaded images)

Method 2: Docker Volume Backup
  $ docker run --rm \
    -v cineshelf_data:/data \
    -v $(pwd):/backup \
    alpine tar czf /backup/cineshelf-backup.tar.gz /data

Restore from backup:
  $ docker run --rm \
    -v cineshelf_data:/data \
    -v $(pwd):/backup \
    alpine sh -c "cd / && tar xzf /backup/cineshelf-backup.tar.gz"
